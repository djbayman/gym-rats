# Gym-Rats

Gym-rats is a fitness website designed to help you get in shape. It offers a
wide range of exercises, step-by-step guides, and comprehensive information
about your body

## Tech Stack:

              - React.js
              - TailwindCss
              - React Query
              - ReRESTful API (from Rapid api)
